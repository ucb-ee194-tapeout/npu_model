# NPU Architecture Spec

This specification document describes the architecture and microarchitecture design of the hardware accelerator for Spring 2026 Tapeout Class. It provides information on the overall device designs and rationales of certain design decisions.
This is a preliminary design specification that is still subject to change. <u>Design choices which are highlighted with underlines in this document represent parameters or configurations that are not fully evaluated or validated</u>. While the development progresses, better understanding and modeling will help us decide on these choices.
